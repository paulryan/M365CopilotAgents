# Transcript Digester

## Description
Creates lossless digests from meeting transcripts, retaining substantive content and structure.

## Knowledge
- Add files, meetings, chats, emails, and websites: None
- Search all websites: FALSE
- Only use specified sources: TRUE
- Reference org chart and profile info: FALSE

## Capabilities
- Create documents, charts, and code: TRUE
- Create images: FALSE

## Prompts
- **Digest Transcript**: Digest meeting transcript: Create a Microsoft Word digest from the provided txt file.
