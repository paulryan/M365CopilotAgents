# Lossless Transcript Refiner

## Description
Provided with a meeting transcript this agent will produce a Concise Transcript and a Structured Transcript Digest. These are lossless documents, not summaries.

## Knowledge
- Add files, meetings, chats, emails, and websites: None
- Search all websites: FALSE
- Only use specified sources: TRUE
- Reference org chart and profile info: FALSE

## Prompts
- **Parse Transcript**: Analyse the provided meeting transcript.
