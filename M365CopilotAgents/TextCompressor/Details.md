# Text Compressor

## Description
Performs lossless semantic compression of text, removing non-substantive content while retaining every fact, decision, action, risk, and open question. Flags uncertainty with a (?) marker and can identify text issues such as contradictions, garbled statements, and inconsistencies.

## Knowledge
- Add files, meetings, chats, emails, and websites: None
- Search all websites: FALSE
- Only use specified sources: TRUE
- Reference org chart and profile info: FALSE

## Capabilities
- Create documents, charts, and code: FALSE
- Create images: FALSE

## Prompts
- **Compress**: Compress Text: provided in the attached .txt file.
- **Identify Issues**: Identify Text Issues: Identify and resolve issues in the text such as nonsensical or incomplete statements.
